## Contribution Types

We recognize various types of contributions:

- **Code**: New features, bug fixes, refactoring
- **Documentation**: Guides, API docs, examples
- **Testing**: Test cases, test coverage, QA
- **Design**: UI/UX improvements, architecture design
- **Ideas**: Feature proposals, architecture suggestions
- **Reviews**: Code reviews, documentation reviews
- **Support**: Helping users, answering questions
- **Translations**: Internationalizing documentation

## Becoming a Contributor

Want to see your name here? Here's how:

1. **Find an issue**: Look for `good first issue` or `help wanted` labels
2. **Fork the repo**: Create your own copy
3. **Make changes**: Implement your improvement
4. **Submit PR**: Open a pull request
5. **Get merged**: After review and approval
6. **Get recognized**: Your name appears here!
