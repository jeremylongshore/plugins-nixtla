# Embedded Forecast Widget - Business Case

**Plugin:** nixtla-embedded-forecast-widget
**Category:** External Revenue
**Status:** Planned
**Last Updated:** 2025-12-15

---

## Problem Statement

SaaS companies want to add forecasting features to their products but lack ML expertise. Building forecasting from scratch takes 6-12 months. These companies would pay for a drop-in solution, but Nixtla's current offering requires significant integration effort.

---

## Target Customer

| User | Need |
|------|------|
| SaaS product managers | Add forecasting features quickly |
| Development teams | Drop-in component, minimal integration |
| Startup founders | Competitive feature without ML team |
| Enterprise software | White-label forecasting capability |

---

## Market Opportunity

- **TAM**: 10,000+ SaaS companies need forecasting
- **Distribution**: Partners embed, Nixtla earns per-forecast
- **Scale**: 1M+ end-user forecasts/month achievable
- **Projected ARR**: $200-500K by year 2

## Competitive Positioning

No competitor offers a white-label, embeddable forecasting widget. Competitors require full API integration, which takes months. The widget takes hours.
