# Documentation Index

## Claude Code Plugins for Nixtla - Internal Documentation

This directory contains all internal documentation, planning materials, architecture decisions, and reference documents for the Claude Code Plugins for Nixtla project.

## Document Naming Standard

All documents follow the format: `NNN-CC-ABCD-description.md`

- **NNN**: Sequence number (001-999)
- **CC**: Category code (PP, AT, RA, DR, etc.)
- **ABCD**: Document type abbreviation
- **description**: Brief description in kebab-case

## Document Categories

| Code | Category | Purpose |
|------|----------|---------|
| PP | Product & Planning | Requirements, roadmaps, planning |
| AT | Architecture & Technical | Technical decisions, system design |
| RA | Reports & Analysis | Research, analytics, intelligence |
| DR | Documentation & Reference | Guides, standards, references |
