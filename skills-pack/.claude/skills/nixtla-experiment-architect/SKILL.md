# Nixtla Experiment Architect

This skill **designs and scaffolds complete forecasting experiments** using Nixtla's StatsForecast, MLForecast, and TimeGPT libraries. It creates production-ready experiment harnesses with configuration management, multi-model comparison, and rigorous evaluation.

## When This Skill Activates

**Trigger phrases**:
- "Set up a forecasting experiment"
- "Compare TimeGPT to baselines"
- "Create a forecasting pipeline"
- "Benchmark models on my data"
- "Design a forecasting workflow"

**File patterns**:
- User asks about experiment setup while in a forecasting project
- Mentions cross-validation, model comparison, or benchmark

## What This Skill Does

1. **Gathers experiment requirements** (data source, target, horizon, frequency)
2. **Creates directory structure** (`forecasting/` if not exists)
3. **Generates configuration file** (`forecasting/config.yml`)
4. **Scaffolds experiment harness** (`forecasting/experiments.py`)
5. **Sets up multi-model comparison** (StatsForecast + MLForecast + TimeGPT)
6. **Configures cross-validation** (rolling-origin or expanding-window)
7. **Handles missing libraries gracefully** (generates TODOs with clear instructions)

### Step 3: Create Configuration File

Generate `forecasting/config.yml` with experiment parameters:

```yaml
# Forecasting Experiment Configuration
# Generated by nixtla-experiment-architect skill

# Data Source
data:
  source: "data/sales.csv"  # Or SQL query, dbt ref, etc.
  unique_id: "store_id"     # Series identifier (null for single series)
  ds: "date"                # Timestamp column
  y: "sales"                # Target column
  exog_vars: []             # Exogenous variables (optional)

# Forecasting Parameters
forecast:
  horizon: 30               # Forecast h steps ahead
  freq: "D"                 # Pandas frequency string
  season_length: 7          # For seasonal models

# Cross-Validation Settings
cv:
  method: "rolling"         # "rolling" or "expanding"
  h: 14                     # Validation horizon
  step_size: 7              # How far to move window
  n_windows: 4              # Number of validation folds

# Models to Evaluate
models:
  statsforecast:
    - SeasonalNaive
    - AutoARIMA
    - AutoETS
    - AutoTheta

  mlforecast:
    enabled: true
    base_models: [RandomForestRegressor, LGBMRegressor]
    lags: [1, 7, 14, 28]
    lag_transforms: true

  timegpt:
    enabled: false          # Set to true if NIXTLA_API_KEY available
    finetune_steps: 0
    level: [80, 90]

# Evaluation Metrics
metrics: [smape, mase, rmse]

# Output Settings
output:
  save_forecasts: true
  save_cv_results: true
  results_dir: "forecasting/results"
```

**Customization based on user input**:
- If single series: set `unique_id: null`
- If hourly data: set `freq: "H"`, `season_length: 24`
- If monthly: set `freq: "M"`, `season_length: 12`
- If TimeGPT mentioned: set `timegpt.enabled: true`

### Step 4: Generate Experiment Harness

Create `forecasting/experiments.py` with complete experiment workflow.

**Key components**:

```python
"""
Nixtla Forecasting Experiment Harness
Generated by nixtla-experiment-architect skill
"""

import os
import yaml
import pandas as pd
from pathlib import Path

# Load configuration
with open('forecasting/config.yml', 'r') as f:
    config = yaml.safe_load(f)

# ========== DATA LOADING ==========

def load_data(config):
    """Load data and transform to Nixtla schema (unique_id, ds, y)"""
    data_cfg = config['data']
    source = data_cfg['source']

    # Detect source type (CSV, Parquet, SQL, dbt)
    if source.endswith('.csv'):
        df = pd.read_csv(source)
    elif source.endswith('.parquet'):
        df = pd.read_parquet(source)
    # ... (see full template for SQL/dbt handling)

    # Transform to Nixtla schema
    df_nixtla = df.rename(columns={
        data_cfg['ds']: 'ds',
        data_cfg['y']: 'y'
    })

    # Handle unique_id
    if data_cfg['unique_id']:
        df_nixtla = df_nixtla.rename(columns={data_cfg['unique_id']: 'unique_id'})
    else:
        df_nixtla['unique_id'] = 'series_1'

    df_nixtla['ds'] = pd.to_datetime(df_nixtla['ds'])
    df_nixtla['y'] = pd.to_numeric(df_nixtla['y'], errors='coerce')
    df_nixtla = df_nixtla.dropna(subset=['y'])

    return df_nixtla[['unique_id', 'ds', 'y']]

# ========== MODEL FUNCTIONS ==========

def run_statsforecast(df, config):
    """Run StatsForecast baseline models with cross-validation"""
    from statsforecast import StatsForecast
    from statsforecast.models import SeasonalNaive, AutoARIMA, AutoETS, AutoTheta

    # Build models from config
    models = [SeasonalNaive(season_length=config['forecast']['season_length']),
              AutoARIMA(), AutoETS(), AutoTheta()]

    sf = StatsForecast(models=models, freq=config['forecast']['freq'], n_jobs=-1)
    sf.fit(df)

    # Cross-validation
    cv_cfg = config['cv']
    cv_df = sf.cross_validation(
        df=df, h=cv_cfg['h'], step_size=cv_cfg['step_size'],
        n_windows=cv_cfg['n_windows'], refit=(cv_cfg['method'] == 'expanding')
    )
    return cv_df

def run_mlforecast(df, config):
    """Run MLForecast with lag features (if enabled)"""
    if not config['models']['mlforecast']['enabled']:
        return None

    from mlforecast import MLForecast
    from mlforecast.lag_transforms import expanding_mean, rolling_std
    from sklearn.ensemble import RandomForestRegressor
    from lightgbm import LGBMRegressor

    ml_cfg = config['models']['mlforecast']
    models = [RandomForestRegressor(n_estimators=100, random_state=42),
              LGBMRegressor(n_estimators=100, random_state=42)]

    mlf = MLForecast(
        models=models, freq=config['forecast']['freq'],
        lags=ml_cfg['lags'],
        lag_transforms={7: [rolling_std], 28: [expanding_mean]} if ml_cfg.get('lag_transforms') else None,
        date_features=['dayofweek', 'month', 'quarter']
    )

    cv_cfg = config['cv']
    cv_df = mlf.cross_validation(
        df=df, h=cv_cfg['h'], step_size=cv_cfg['step_size'],
        n_windows=cv_cfg['n_windows'], refit=(cv_cfg['method'] == 'expanding')
    )
    return cv_df

def run_timegpt(df, config):
    """Run TimeGPT (if API key configured)"""
    if not config['models']['timegpt']['enabled']:
        return None

    api_key = os.getenv('NIXTLA_API_KEY')
    if not api_key:
        print("⚠️  Set NIXTLA_API_KEY to enable TimeGPT")
        return None

    from nixtla import NixtlaClient
    client = NixtlaClient(api_key=api_key)

    tg_cfg = config['models']['timegpt']
    cv_cfg = config['cv']

    cv_df = client.cross_validation(
        df=df, h=cv_cfg['h'], n_windows=cv_cfg['n_windows'],
        step_size=cv_cfg['step_size'], level=tg_cfg.get('level', [80, 90]),
        finetune_steps=tg_cfg.get('finetune_steps', 0)
    )
    return cv_df

# ========== EVALUATION ==========

def evaluate_models(cv_results, config):
    """Compute evaluation metrics across all models"""
    from utilsforecast.evaluation import evaluate
    from utilsforecast.losses import smape, mase, rmse

    # Combine all CV results
    all_cv = [cv_df for cv_df in cv_results.values() if cv_df is not None]
    if not all_cv:
        return None

    combined_cv = pd.concat(all_cv, axis=0).reset_index(drop=True)

    metric_map = {'smape': smape, 'mase': mase, 'rmse': rmse}
    metrics = [metric_map[m] for m in config['metrics'] if m in metric_map]

    metrics_df = evaluate(combined_cv, metrics=metrics, models=list(cv_results.keys()))
    return metrics_df

# ========== MAIN ==========

def main():
    print("🔬 Nixtla Forecasting Experiment\n")

    # Load data
    df = load_data(config)
    print(f"Loaded {len(df)} rows, {df['unique_id'].nunique()} series\n")

    # Run models
    cv_results = {}
    if config['models']['statsforecast']:
        cv_results['StatsForecast'] = run_statsforecast(df, config)
    if config['models']['mlforecast']['enabled']:
        cv_results['MLForecast'] = run_mlforecast(df, config)
    if config['models']['timegpt']['enabled']:
        cv_results['TimeGPT'] = run_timegpt(df, config)

    # Evaluate
    metrics_df = evaluate_models(cv_results, config)
    if metrics_df is not None:
        print("\n📈 RESULTS")
        print(metrics_df.groupby('model').mean())

        # Save results
        results_dir = Path(config['output']['results_dir'])
        results_dir.mkdir(parents=True, exist_ok=True)
        metrics_df.to_csv(results_dir / "metrics_summary.csv", index=False)
        print(f"\n💾 Saved: {results_dir}/metrics_summary.csv")

    print("\n✅ Experiment complete!")

if __name__ == "__main__":
    main()
```
